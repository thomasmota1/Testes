## Padrao de Organizacao

Cada aula fica em uma pasta propria dentro de `aulas/`.

Quando houver atividade escrita, ela deve ficar no `README.md` da aula. Quando houver imagens ou prints, eles devem ficar em `imagens/`. Quando houver codigo, ele deve ficar junto da aula, em uma subpasta com nome claro para o laboratorio ou exercicio.
